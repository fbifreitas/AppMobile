## Backend
- Testes unitarios/integracao relevantes executados.
- Contratos/headers/contexto validados.
- Mudancas de modelo refletidas em docs/backlog.
- Evidencia: comando + resultado + risco residual.

## Web/Backoffice
- Lint/test/build executados quando aplicavel.
- Contrato com backend preservado.
- Backlog web atualizado se houver impacto.
- Evidencia: comandos + arquivos alterados.

## Integracao
- Contratos e compatibilidade verificados.
- Idempotencia/context envelope revisados.
- Observabilidade minima considerada.
- Evidencia: validacao de contrato + docs atualizados.

## Documentacao
- Indices e ponteiros atualizados.
- Sem contradicao com V2 ativa.
- Escopo (estrategico/operacional/legado) explicito.

## Backlog/Governanca
- Item rastreavel com status claro.
- Criterio de pronto explicito.
- Dependencias e testes obrigatorios registrados.
- Evidencia: secao do backlog atualizada.
