# PORTFOLIO VIEW

## Product portfolio model

O portfólio passa a ser descrito como uma combinação de plataforma + domain packs + canais.

### Shared platform
- Platform Core
- Shared Foundations
- Data & Intelligence

### Domain packs
- Inspection
- Wellness
- Church

### Channel products
- mobile apps
- web backoffices
- admin portals
- APIs públicas
- integrações/parceiros

## Packaging logic

Um tenant pode contratar:
- apenas um domínio
- múltiplos domínios
- apenas tecnologia
- tecnologia + operação intermediada
- tecnologia + marca própria
