# PRD — Plataforma

## Objetivo do MVP
Receber demanda, criar case/job, executar vistoria, processar valuation, gerar laudo e devolver status/resultado.

## Escopo funcional
### Mobile
- login;
- lista de jobs;
- aceite;
- check-in;
- captura;
- revisão;
- submissão;
- sync offline.

### Web
- consulta de jobs;
- intake;
- valuation básico;
- emissão de laudo;
- governança básica.

### Backend
- integração inicial;
- APIs internas;
- modelo canônico;
- persistência;
- storage;
- eventos.
