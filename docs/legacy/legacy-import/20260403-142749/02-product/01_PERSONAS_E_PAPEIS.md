# Personas e Papéis

| Persona | Papel |
|---|---|
| Demandante da financeira | Origina a solicitação de vistoria ou avaliação |
| Analista da financeira | Acompanha status, SLA, evidências e resultado |
| Gestor da financeira | Monitora carteira, prazo e qualidade |
| Admin da empresa de avaliação | Configura operação, políticas e usuários |
| Coordenador operacional | Organiza fila e acompanha execução diária |
| Dispatcher | Distribui jobs conforme prioridade e região |
| Analista de intake | Valida consistência inicial da inspeção |
| Analista de comparáveis | Complementa dados de mercado |
| Revisor técnico | Revisa tecnicamente a avaliação |
| Signatário técnico | Assina e assume responsabilidade pelo laudo |
| Vistoriador | Executa a vistoria em campo |
| Coordenador regional | Acompanha rede de campo |
| Admin da plataforma | Gerencia tenants, parâmetros e integrações |
| Suporte da plataforma | Trata incidentes e exceções |
| Admin IAM | Gerencia autenticação e acesso |
| Auditor | Verifica trilha e conformidade |
| Gestor financeiro | Acompanha settlement e resultado financeiro |
| Gestor de marketplace | Acompanha equilíbrio entre demanda e oferta |
| Consumidor do resultado | Recebe laudo, evidência e status |
