# Modelo de Estados Críticos

## Job
Created → EligibleForDispatch → Offered → Accepted → InExecution → FieldCompleted → Submitted → Closed

## Inspection
Draft → InProgress → CompletedLocal → SyncPending → Submitted → Accepted/Rejected

## ValuationProcess
PendingIntake → IntakeValidated → Enriched → Processing → Completed → Approved

## Report
Draft → Generated → UnderReview → ReadyForSign → Signed → Published → Archived
