# Modelo Operacional

## Consoles
- Ops Console
- Technical Console
- Admin Console

## Fluxo principal
1. parceiro cria demanda;
2. integração normaliza;
3. case/job é criado;
4. trabalho é distribuído;
5. vistoria é executada;
6. inspeção é enviada;
7. valuation é processado;
8. laudo é emitido;
9. settlement é calculado.
