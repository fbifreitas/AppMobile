# Checklist de validação do app

## Home
- [ ] abre sem tela branca
- [ ] mostra cabeçalho
- [ ] mostra status de startup
- [ ] mostra card de localização
- [ ] mostra jobs
- [ ] mostra propostas

## Localização
- [ ] botão `Atualizar` funciona
- [ ] latitude/longitude aparecem
- [ ] última atualização aparece
- [ ] distância nos jobs é exibida

## Jobs
- [ ] `COMO CHEGAR` responde
- [ ] `INICIAR VISTORIA` responde
- [ ] labels de distância e raio aparecem

## Navegação
- [ ] notificações abre
- [ ] settings abre
- [ ] hub abre

## Hub operacional
- [ ] abre
- [ ] itens aparecem
- [ ] operação de campo abre sem exceção
- [ ] outras centrais respondem

## Check-in
- [ ] inicia fluxo
- [ ] segue para próxima etapa
- [ ] não trava

## Câmera e classificação
- [ ] câmera abre
- [ ] fluxo principal responde
- [ ] labels e contexto aparecem corretamente

## Estabilidade geral
- [ ] sem travamento
- [ ] sem tela branca
- [ ] sem erro visual crítico
