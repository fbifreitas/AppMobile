# Identity, Access e User Management

## Princípio
User management é capability de plataforma composta por identidade, vínculo organizacional, autorização e elegibilidade operacional.

## Submódulos
- Identity
- Person/Profile
- Tenant & Organization Membership
- Authorization
- Workforce Qualification

## Regra
Usuário é identidade.  
Acesso é contexto.  
Operação é elegibilidade.
