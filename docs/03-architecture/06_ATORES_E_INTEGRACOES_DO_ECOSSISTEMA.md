# Atores e Integrações do Ecossistema

| Ator | Tipo de integração esperada |
|---|---|
| Financeiras | Envio de demanda, consulta de status, laudos e eventos |
| Bancos / originadores | Criação de demanda e acompanhamento |
| Empresas de avaliação | Operação de processos, consumo de resultados |
| Vistoriadores | Integração via app mobile |
| Parceiros de assinatura | Assinatura de laudos |
| Provedores de identidade | SSO e autenticação |
| Gateways de notificação | E-mail, push, SMS |
| Gateways de cobrança | Cobrança, repasse e settlement |
| ERP / financeiro | Conciliação e integração contábil |
| CRM / atendimento | Consulta de status e histórico |
| Storage / documentos | Evidências, anexos e laudos |
| Dados de mercado | Comparáveis e preços |
| Geolocalização / mapas | Região, distância e cobertura |
| Observabilidade | Logs, métricas e traces |
| BI / analytics | Consumo de dados consolidados |
| Auditoria e compliance | Trilhas e evidências |
| Parceiros white label | Integração dedicada |
| Outras empresas de avaliação | Operação como tenants/providers |
